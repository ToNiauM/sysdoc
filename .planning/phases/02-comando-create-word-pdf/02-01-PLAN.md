# Plan 02-01: Implementar `sysdoc create` para gerar TR `.docx`

<objective>
Implementar o comando `sysdoc create [pasta] [tipo]` para gerar, de forma offline e determinística, um Termo de Referência `.docx` a partir de `dados_consolidados.json`, `.sysdoc/cache/textos/ETP.txt` e um template Word fixo do projeto. O MVP gera apenas TR (`tipo=tr`) como base para edição humana.
</objective>

<context>
Phase 2 já tem escopo definido em `02-CONTEXT.md`: gerar apenas TR `.docx`, como base para edição humana, usando cabeçalho do JSON consolidado e corpo do ETP literal, com cláusulas problemáticas do ETP substituídas pelo campo `para`. Itens do TR ficam fora do MVP.
</context>

<truths>
- A CLI do SysDoc permanece offline e determinística; `create` não chama LLM.
- O comando deve aceitar o macro documentado: `/sysdoc create [pasta] [tipo]`.
- O MVP aceita apenas `tr`.
- O corpo-base vem de `.sysdoc/cache/textos/ETP.txt`.
- As substituições aplicam apenas itens com `documento == "ETP"` e `classificação in {"ajuste_necessário", "risco"}`.
- Substituição deve ser exata e conservadora: se `de` não for encontrado, não fazer substituição fuzzy; registrar pendência no `.docx`.
- Não modificar `templates/analise_template.html`, `templates/render_analise.py` nem `templates/validate_sysdoc.py`.
</truths>

<threat_model>
**Assets:** documentos públicos de contratação, JSON de análise, texto extraído do ETP e TR gerado em Word.

**Threats considered:**
- Substituir a cláusula errada por correspondência fuzzy em texto extraído de PDF.
- Gerar documento com placeholders não substituídos.
- Sobrescrever um `.docx` anterior sem versionamento.
- Criar falsa sensação de documento final oficial.

**Mitigations required in this plan:**
- Usar substituição exata de `de` por `para`.
- Criar seção "Substituições pendentes" quando uma substituição não for aplicada.
- Auto-incrementar nomes `tr_[modelo]_[data].docx`, `_2`, `_3`.
- Mensagens e documentação devem declarar que o `.docx` é base para edição humana.
</threat_model>

7. Adicionar:

```python
def apply_etp_revisions(etp_text: str, items: list[dict]) -> tuple[str, list[dict]]:
    revised = etp_text
    pending = []
    for item in items:
        before = str(item.get("de") or "")
        after = str(item.get("para") or "")
        if before and after and before in revised:
            revised = revised.replace(before, after, 1)
        else:
            pending.append(item)
    return revised, pending
```

8. Adicionar:

```python
def resolve_next_docx_output(project_dir: Path, prefix: str, model: str, date: str) -> Path:
    stem = f"{prefix}_{model}_{date}"
    candidate = project_dir / f"{stem}.docx"
    if not candidate.exists():
        return candidate
    index = 2
    while True:
        candidate = project_dir / f"{stem}_{index}.docx"
        if not candidate.exists():
            return candidate
        index += 1
```

def write_tr_docx(paths: ProjectPaths, data: dict, revised_etp: str, pending: list[dict], output: Path) -> None:
    try:
        from docx import Document
    except ImportError as exc:
        raise RuntimeError("python-docx não está instalado; instale com: pip install python-docx") from exc
    if not paths.tr_template.is_file():
        raise SystemExit(f"Template TR não encontrado: {rel(paths.tr_template)}")
    document = Document(str(paths.tr_template))
    projeto = data.get("projeto", {})
    values = {
        "{{objeto}}": str(projeto.get("objeto") or "não informado"),
        "{{processo}}": str(projeto.get("processo") or "não informado"),
        "{{valor_estimado}}": str(projeto.get("valor_estimado") or "não informado"),
        "{{orgao}}": str(projeto.get("órgão") or projeto.get("orgao") or "não informado"),
        "{{data_analise}}": str(data.get("data_análise") or "não informada"),
        "{{modelo_ia}}": str(data.get("modelo_ia") or "não informado"),
        "{{corpo_etp}}": revised_etp,
        "{{substituicoes_pendentes}}": build_pending_text(pending),
    }
    _replace_docx_placeholders(document, values)
    document.save(str(output))

Implementar:

```python
def create(project: str, tipo: str = "tr") -> int:
    if tipo != "tr":
        raise SystemExit(f"Tipo não suportado nesta versão: {tipo}")
    paths = project_paths(project)
    data = load_analysis_json(paths)
    etp_path = ensure_etp_text(project, paths)
    etp_text = etp_path.read_text(encoding="utf-8", errors="replace")
    revised, pending = apply_etp_revisions(etp_text, applicable_etp_items(data))
    model = slug(data.get("modelo_ia", "modelo"))
    date = extract_date(data.get("data_análise", ""))
    output = resolve_next_docx_output(paths.root, "tr", model, date)
    write_tr_docx(paths, data, revised, pending, output)
    print(f"TR gerado: {rel(output)}")
    if pending:
        print(f"Substituições pendentes: {len(pending)}")
    return 0
```
